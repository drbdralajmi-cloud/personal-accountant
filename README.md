# 💸 المحاسب الشخصي — Personal Accountant

تطبيق ويب بسيط وسهل لمتابعة الأموال، بتصميم احترافي ودعم كامل للعربية والإنجليزية.

A simple, professional personal finance tracker — bilingual (Arabic / English), works fully offline.

## ✨ الميزات | Features

- 💰 **تسجيل المصروفات والدخل** — مع المبلغ والفئة والتاريخ والملاحظة.
- 🏷️ **تصنيف الفئات** — فئات جاهزة + إضافة فئات مخصّصة (أيقونة ولون).
- 🎯 **ميزانية شهرية** — تحديد سقف للصرف مع تنبيه عند الاقتراب أو التجاوز.
- 📊 **تقارير ورسوم بيانية** — مخطط الصرف حسب الفئة، والدخل مقابل المصروفات عبر آخر ٦ أشهر.
- 🌐 **عربي / إنجليزي** — مع دعم اتجاه RTL، واختيار العملة (ريال، درهم، دولار...).
- 🌙 **وضع داكن / فاتح / تلقائي**.
- 💾 **يعمل بدون إنترنت** — البيانات تُحفظ محلياً على جهازك (localStorage)، مع تصدير/استيراد.

## 🚀 التشغيل | Run locally

التطبيق ملفات ثابتة بدون خطوة بناء. شغّل أي خادم ملفات بسيط:

```bash
python3 -m http.server 8000
```

ثم افتح: http://localhost:8000

## 🛠️ التقنيات | Tech

- HTML / CSS / JavaScript (ES Modules) — بدون أُطر أو خطوة بناء.
- [Chart.js](https://www.chartjs.org/) للرسوم البيانية (محمّلة محلياً).
- التخزين عبر `localStorage`.

## 📂 البنية | Structure

```
index.html        # هيكل الصفحة
styles.css        # التصميم (ثيم فاتح/داكن + RTL)
js/
  app.js          # المنطق والشاشات والتنقّل

  i18n.js         # الترجمة عربي/إنجليزي
  charts.js       # الرسوم البيانية
vendor/
  chart.umd.min.js
```
